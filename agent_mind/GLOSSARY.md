# Глоссарий

Устойчивые термины WarSim. Канонические определения — в `knowledge_base/`; здесь — кратко для агента.

| Термин | Значение |
|--------|----------|
| **FMP** | Formation Material Physics — бой как физика материала формации |
| **Formation Field** | Слой 1: формация как поле, бойцы — точки выборки |
| **Contact Zone** | Слой 2: полная физика на границе двух формаций (передний ряд) |
| **Individual Physics** | Слой 3: полная физика для героя и событийных бойцов |
| **GIC** | Geometric Impulse Computation — импульс удара из геометрии shapecast |
| **Stiffness** | Жёсткость материала ↔ **организованность** |
| **Yield strength** | Предел текучести ↔ **мораль** (и усталость) |
| **Viscosity** | Вязкость / восстановление ↔ опыт, «текучесть» деморализованной массы |
| **Anisotropy** | Разные свойства по осям ↔ тип **формации** (клин жёсток вдоль атаки) |
| **Моральный слом** | Давление > yield → боец **бежит**, паника по волне (слой 1) |
| **Физический выбой** | Смещение > порога организованности → боец **вне слота**, но может сражаться (слой 3) |
| **Отряд** | Организованная боевая группа, не набор HP-юнитов |
| **Формация** | Режим коллективного поведения (движение, плотность, уязвимости) |
| **Герой** | Часть строя, моральный центр, источник приказов; всегда слой 3 |
| **FMP-лаборатория** | Минимальный runnable для проверки FMP, не полная игра |
| **Канон** | `knowledge_base/` — источник истины по дизайну и FMP |
| **Agent Mind** | `agent_mind/` — оперативная память агента |

См. также: [`FMP.md`](../knowledge_base/FMP.md), [`PROJECT_AXIOMS.md`](../knowledge_base/PROJECT_AXIOMS.md).
